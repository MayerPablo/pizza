# Pablo and Felix' Pizza review site
We rate pizza restaurants on this site.

There are 3 simple rules:
1. Every Pizza gets <b>2 ratings from .0 to 10.0</b>, one for the general quality of the pizza (GQ) and one for the value offered (VO)
2. The rating scale is normalized to the Freiburger Mensa Institutsviertel Buffet Pizza which is evaluated at a
    - <b>General quality (GQ): 5.0/10</b> 
    - <b>Value offered (VO): 7.0/10</b>  (5.0€ Edit: 5.9€ as of 01.11.2025).
3. The ambience of the restaurant, friendliness of the staff, options on the menu, blablabla, [...] <b><ins>do not have any influence</ins></b> on these ratings!
